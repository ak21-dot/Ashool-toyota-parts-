# نظام محلات طيب علي صالح الأشول V3.0

## بيانات الدخول
- **المدير**: كلمة المرور `2005`
- **الموظف**: كلمة المرور `0000`

## الصلاحيات
- المدير: كل العمليات + مزامنة استقبال.
- الموظف: مبيعات، ديون، عهد، عمال (عرض)، تصدير يومي.

## المزامنة
1. في نهاية اليوم، الموظف يضغط "تصدير بيانات اليوم".
2. يرسل ملف JSON إلى المدير.
3. المدير يفتح "مزامنة" > "استيراد ودمج".

## التقرير الشهري
- المدير يضغط "تقرير شهري PDF" لحفظ نسخة احتياطية شاملة.

## الأمان
- كلمات المرور مشفرة بـ SHA-256.
- يُنصح بتغييرها من داخل الكود (ملف app.js) قبل النشر.
